
# CineMojo - C# Windows Forms Application

## How to Run
1. Ensure you have **Visual Studio 2022** installed with the **.NET Desktop Development** workload.
2. Ensure you have the **.NET 8.0 SDK** installed.
3. Open the `MovieSuggestionApp.csproj` file in Visual Studio.
4. Press **F5** or click the **Start** button to build and run.

## Troubleshooting
- If you get errors about missing namespaces, ensure you are targeting .NET 8.0 or later.
- If you are on Mac or Linux, this application will NOT run natively as it uses Windows Forms (Windows only).
